ChambaYa - App para Clientes

(Proyecto Académico) | Ver en GitHub

Tecnologías: Kotlin, Google Maps SDK, XML, Jetpack Compose

Funcionalidades Desarrolladas:

  • Diseñé e implementé la interfaz de usuario (XML/Jetpack) para el flujo completo de registro, login y visualización de servicios.
  • Implementé un sistema de autenticación de usuarios para la gestión de perfiles y sesiones seguras.
  • Desarrollé la integración para consumir una API REST y mostrar en tiempo real la disponibilidad de trabajadores y el estado de las solicitudes.

Propósito: Aplicación móvil que permite a los usuarios solicitar servicios domésticos y de mantenimiento de manera rápida y segura, con seguimiento en tiempo real del estado de sus solicitudes.

ChambaYa - App para Trabajadores

(Proyecto Académico) | Ver en GitHub

Tecnologías: Kotlin, Google Maps SDK, Notificaciones Push

Funcionalidades Desarrolladas:

  • Creé la funcionalidad de registro y autorización de roles para trabajadores, asegurando el acceso correcto a la plataforma.
  • Implementé la lógica de negocio para la asignación y seguimiento de trabajos en tiempo real mediante el consumo de una API REST.
  • Integré Google Maps SDK para la geolocalización de trabajadores y configuré notificaciones push para nuevas solicitudes de servicio.

Propósito: Aplicación complementaria que permite a los trabajadores recibir, gestionar y completar solicitudes de servicios, con geolocalización y notificaciones en tiempo real para optimizar su productividad.

Astromarket: Plataforma Web Marketplace

(Proyecto Académico) | Ver en GitHub

Tecnologías: JavaScript, Node.js, Express.js, PostgreSQL, HTML5, CSS3

Funcionalidades Desarrolladas:

  • Diseñé e implementé el esquema de la base de datos en PostgreSQL, estableciendo la arquitectura de datos para la plataforma.
  • Desarrollé una API RESTful completa con Node.js y Express para gestionar las operaciones CRUD (Crear, Leer, Actualizar, Eliminar) de productos.
  • Construí una interfaz de usuario responsiva y dinámica utilizando HTML, CSS y JavaScript para interactuar con la API, permitiendo la visualización y búsqueda de productos.

Propósito: Plataforma de marketplace web que conecta vendedores y compradores, proporcionando un espacio seguro para la compra-venta de productos con funcionalidades de búsqueda avanzada y gestión de inventario.
